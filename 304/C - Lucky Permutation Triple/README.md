For example, <span class="tex-span">[0, 2, 1]</span> is a permutation of length 3 while both <span class="tex-span">[0, 2, 2]</span> and <span class="tex-span">[1, 2, 3]</span> is not.</p><p>A permutation triple of permutations of length <span class="tex-span"><i>n</i></span> <span class="tex-span">(<i>a</i>, <i>b</i>, <i>c</i>)</span> is called a Lucky Permutation Triple if and only if <img align="middle" class="tex-formula" src="https://espresso.codeforces.com/a5c95acda2f8bfd5804d6d872b623ebd819b0d2a.png" style="max-width: 100.0%;max-height: 100.0%;">.
